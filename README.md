# kosand
Tool for managing kosandr sites and apps.

Usage
```bash

kosand start [foreground/fg]
   start daemon
kosand stop
   stop background daemon

kosand restart

kosand setup

kosand init <sitename>
   new website
kosand new-app <appname>
   make new app
kosand install-app

kosandr prompt on
kosandr prompt off

```
